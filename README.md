# Dynamic Credit Pricing using Reinforcement Learning and Bellman Optimisation

This project develops a reinforcement learning–based framework for optimal loan pricing under stochastic credit risk conditions. 

Using a Markov Decision Process and Bellman optimisation, the model dynamically adjusts interest rates by accounting for borrower default, prepayment behaviour, and evolving macroeconomic factors. 

The approach integrates data preprocessing, simulation, and policy learning using different techniques to identify pricing strategies that balance profitability with risk exposure. 

Implemented in Python, the project demonstrates how machine learning techniques can enhance traditional credit modelling frameworks. The results highlight the potential of adaptive pricing systems in improving risk-adjusted returns in lending and structured credit environments, aligning with modern quantitative finance practices.
